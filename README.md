# Hackathon I: Physical AI & Humanoid Robotics Textbook with Embedded RAG Chatbot

This is a textbook for teaching a course in Physical AI & Humanoid Robotics. The
book is developed using Docusaurus and is published on GitHub Pages.

The book further features an embedded RAG chatbot, used for answering the user's
questions about the book, and also including the feature to answer questions
solely based on text selected by the user.

## Tech Stack Used

### Book
- Development: Docusaurus
- Publishing: GitHub Pages

### RAG Chatbot
- Chatbot AI: OpenAI Agents SDK
- Webserver API: FastAPI
- Database: Neon Serverless Postgres Free Tier
- Vector Database and Search Engine: Qdrant Cloud Free Tier

The textbook is based on the course details mentioned in the file
`course-details.md`.
