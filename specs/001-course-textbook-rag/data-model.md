# Data Model: Physical AI & Humanoid Robotics Course Textbook with RAG Chatbot

**Feature Branch**: `001-course-textbook-rag`  
**Created**: 2026-04-18  
**Purpose**: Define the core entities, their attributes, and relationships for the textbook and RAG chatbot system.

## Entities

### 1. Course Textbook

*   **Description**: The primary container for all course content.
*   **Attributes**:
    *   `id` (UUID): Unique identifier for the textbook.
    *   `title` (String): "Physical AI & Humanoid Robotics".
    *   `version` (String): Version of the textbook content (e.g., "1.0.0").
    *   `last_updated` (DateTime): Timestamp of the last content update.
    *   `content_path` (String): File system path or URL to the root of the Docusaurus content.

### 2. Course Module

*   **Description**: A logical section or chapter within the Course Textbook.
*   **Attributes**:
    *   `id` (UUID): Unique identifier for the module.
    *   `textbook_id` (UUID): Foreign key referencing the Course Textbook.
    *   `title` (String): Title of the module (e.g., "The Robotic Nervous System (ROS 2)").
    *   `order` (Integer): Display order within the textbook.
    *   `content` (Text): The actual markdown content of the module.
    *   `slug` (String): URL-friendly identifier for the module.
*   **Relationships**: Many-to-one with Course Textbook.

### 3. RAG Chatbot Knowledge Base (Implicit)

*   **Description**: The indexed and vectorized representation of the Course Textbook content, stored in Qdrant for retrieval.
*   **Attributes** (managed by Qdrant/OpenAI Agents SDK):
    *   `vector_embedding` (Vector): Numerical representation of text segments.
    *   `original_text` (Text): The source text segment.
    *   `source_module_id` (UUID): Reference to the Course Module the text originated from.
    *   `page_number` (Integer): Optional, page number within the module.

### 4. Student Query

*   **Description**: A natural language question posed by a student to the RAG Chatbot.
*   **Attributes**:
    *   `id` (UUID): Unique identifier for the query.
    *   `timestamp` (DateTime): Time when the query was made.
    *   `query_text` (String): The student's question.
    *   `context_text` (String, Optional): The selected text segment provided by the student.
    *   `response_id` (UUID, Optional): Foreign key referencing the Chatbot Response.
*   **Relationships**: One-to-one with Chatbot Response.

### 5. Chatbot Response

*   **Description**: The answer generated by the RAG Chatbot in response to a Student Query.
*   **Attributes**:
    *   `id` (UUID): Unique identifier for the response.
    *   `timestamp` (DateTime): Time when the response was generated.
    *   `response_text` (String): The chatbot's answer.
    *   `source_references` (List of Strings, Optional): List of references (e.g., module titles, page numbers) from where information was retrieved.
    *   `query_id` (UUID): Foreign key referencing the Student Query.
*   **Relationships**: One-to-one with Student Query.

## Relationships Summary

*   A **Course Textbook** contains many **Course Modules**.
*   **Course Modules** provide the content that is indexed into the **RAG Chatbot Knowledge Base**.
*   A **Student Query** is answered by a **Chatbot Response**.
*   A **Chatbot Response** is linked to a **Student Query**.
*   The **RAG Chatbot Knowledge Base** implicitly references **Course Modules** through `source_module_id`.
